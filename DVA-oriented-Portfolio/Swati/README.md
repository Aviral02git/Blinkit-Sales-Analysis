# Swati Ghosh DVA Portfolio

Static HTML and CSS portfolio for Data Visualization & Analytics case studies.

## Files

- `index.html` - portfolio homepage.
- `project-loan-default.html` - loan default analytics case study.
- `project-blinkit.html` - Blinkit analytics case study.
- `project-uber.html` - Uber trips analytics case study.
- `styles.css` - shared visual styling.
- `images/` - dashboard screenshots used by the portfolio.
- `reports/` - local report PDFs linked from project pages.
